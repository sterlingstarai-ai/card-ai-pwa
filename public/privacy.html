개인정보처리방침

시행일: 2026년 1월 18일

1. 수집하는 개인정보

Card AI는 서비스 제공을 위해 최소한의 정보만 처리합니다.

  • 수집하지 않는 정보: 카드번호, 주민등록번호, 전화번호, 정확한 위치 좌표
  • 기기 내 저장 정보: 선택한 카드 목록, 최근 방문 장소 (서버 전송 없음)
  • 제보 시 수집 정보: 카드명, 장소명, 혜택 내용 (GitHub Issues로 전송)

2. 개인정보의 처리 목적

  • 카드 혜택 추천 서비스 제공
  • 데이터 오류 수정 및 서비스 개선

3. 개인정보의 보유 및 이용 기간

사용자가 선택한 카드 정보는 기기 내에만 저장되며, 앱 삭제 시 함께 삭제됩니다. 제보 데이터는 GitHub Issues에 공개적으로 기록됩니다.

4. 개인정보의 제3자 제공

Card AI는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 데이터 제보 시 입력한 내용은 GitHub에 공개됩니다.

5. 개인정보의 파기

앱 삭제 시 기기에 저장된 모든 데이터가 자동으로 삭제됩니다.

6. 정보주체의 권리

사용자는 언제든지 앱 내 설정에서 저장된 데이터를 초기화할 수 있습니다.
